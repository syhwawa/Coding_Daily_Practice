Given an integer (signed 32 bits), write a function to check whether it is a power of 4.

Example 1:

Input: 16
Output: true
Example 2:

Input: 5
Output: false
Follow up: Could you solve it without loops/recursion?

```
class Solution:
    def isPowerOfFour(self, num: int) -> bool:
        if num == 0:
            return False
        while num > 0:
            if num % 4 == 0:
                num /= 4
            else:
                break
        return num == 1       
 ```
